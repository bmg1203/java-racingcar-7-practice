# java-racingcar-precourse
## 구현 기능 

### 1. 사용자의 문자열을 입력 받아야 함
- 쉼표로 자동차 이름을 구분함
- 자동차의 이름은 5자 이하로 제한 되어야함
- 자동차 이름은 빈 문자열이나, 공백 문자열이면 안됨
### 2. 사용자의 시도 횟수를 입력 받아야 함 
- 시도 횟수는 숫자여야함
### 3. 각 이동 마다 이동 결과를 출력해야 함
- 각 자동차는 0~9 사이의 랜덤값 중 4 이상일 시 전진할 수 있음
- 전진 길이는 자동차 명과 함께 ```-``` 로 표시함
### 4. 최종 우승자를 출력해야 함
- 최종 우승자는 여러명 일수 있으며, 쉼표로 구분함
### 5. 사용자가 잘못된 값을 입력할 경우 ```IllegalArgumentException``` 을 발생시킨 후 종료되어야 함

## 테스트 케이스

### 성공
1. 시도 횟수가 0 : pobi, 0
2. 시도 횟수가 다수 : pobi, 3
3. 단독 우승 : pobi, woni, jun, 5 
4. 공동 우승 (pobi와 jun이 같은 길이) : pobi, woni, jun, 5 

### 실패
1. 6자 이상의 자동차 이름 입력 : pobi,javaji 1
2. 쉼표가 아닌 구분자로 제한 길이 초과: pobi&jun 1
<br>만약 제한 길이를 초과하지 않았다면, 하나의 이름으로 판단
3. 이름이 빈 문자열 :  1
4. 이름이 공백 문자열 :   1
5. 시도 횟수가 숫자가 아님 : pobi A
6. 자동차 이름이 중복됨 : pobi,pobi

## 기능 요구 사항

초간단 자동차 경주 게임을 구현한다.

- 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
- 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
- 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하만 가능하다.
- 사용자는 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
- 전진하는 조건은 0에서 9 사이에서 무작위 값을 구한 후 무작위 값이 4 이상일 경우이다.
- 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다. 우승자는 한 명 이상일 수 있다.
- 우승자가 여러 명일 경우 쉼표(,)를 이용하여 구분한다.
- 사용자가 잘못된 값을 입력할 경우 `IllegalArgumentException`을 발생시킨 후 애플리케이션은 종료되어야 한다.

### 입출력 요구 사항

#### 입력

- 경주할 자동차 이름(이름은 쉼표(,) 기준으로 구분)
