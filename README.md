# 기능 구현 목록

### 1. 경주할 자동차 이름 입력 기능
- 자동차 이름 문자열을 입력받는 기능
- 입력받은 문자열을 쉼표를 기준으로 구분하는 기능
- 각 자동차 이름이 유효한 이름인지 검증하는 기능

### 2. 자동차 생성 기능
- 각 자동차에 이름을 부여하여 생성

### 3. 시도 횟수 입력 기능
- 사용자가 시도할 횟수를 입력받음

### 4. 자동차 전진 기능
- 전진 조건: 0~9 사이의 무작위 값이 4 이상일 때 전진

### 5. 우승자 확인 기능
- 우승자 출력: 우승자가 여러 명일 경우 쉼표(`,`)로 구분하여 표시
