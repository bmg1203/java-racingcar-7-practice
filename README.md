# [우아한테크코스(7기)] 프리코스 2주차 - 자동차 경주
## 문제
    간단한 자동차 경주 게임을 구현한다
## 구현할 기능
- 사용자 입력 (자동차 이름, 횟수)
- 결과 출력 (차수별 결과, 우승자 안내)
- 각 자동차의 이동 가능 여부 판별
- 우승자 판별
- 잘못된 입력값에 대한 예외 처리 (IllegalArgumentException)
## 입출력 예시
- 자동차 이름 입력(','로 구분)

      pobi,woni,jun
- 시행 횟수 입력

      5
- 차수별 실행 결과 출력

      pobi : --
      woni : ----
      jun : ---
- 우승자 안내 출력

      최종 우승자 : pobi
- 최종 실행 결과

      경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)
      pobi,woni,jun
      시도할 횟수는 몇 회인가요?
      5
    
      실행 결과
      pobi : -
      woni :
      jun : -
    
      pobi : --
      woni : -
      jun : --
    
      pobi : ---
      woni : --
      jun : ---
    
      pobi : ----
      woni : ---
      jun : ----
    
      pobi : -----
      woni : ----
      jun : -----
    
      최종 우승자 : pobi, jun
