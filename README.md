# java-racingcar-precourse

### 기능 목록

-[ ] 자동차는 4이상일 경우 이동 한다.
-[ ] 이름을 ,로 분리한다.
-[ ] 0~9사이의 랜덤한 숫자를 생성한다
-[ ] 이동 횟수 만큼 자동차를 이동한다

### ui 기능목록

-[ ] 몇 번 이동할 것인지 입력받는다.
-[ ] 자동차의 이름들을 입력받는다.
-[ ] 이동 결과를 출력한다.(이름 : -(이동횟수만큼 표시))
-[ ] 우승자를 출력한다.(최종 우승자 : pobi, jun)

### 예외 요구 기능목록

-[x] 이름이 6자 이상이면 예외를 발생한다.
-[x] 이름이 비어 있을 시 예외를 발생한다.
