## 처리할 예외 상황 목록

- 자동차 이름으로 "" 혹은 " "가 입력된 경우
  - `IllegalArgumentException`을 반환한다.
- 시도 횟수가 음수인 경우
  - `IllegalArgumentException`을 반환한다.
