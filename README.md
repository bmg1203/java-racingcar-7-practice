# java-racingcar-precourse

## 구현할 기능

### 입력 처리

- 자동차 이름 입력
- 시도할 횟수 입력

### 핵심 로직

- 입력 문자열 파싱
    - 입력된 문자열 -> 자동차 이름들
    - 쉼표(`,`) 기준, 5글자 이하
- 입력된 횟수 만큼 전진/이동
    - 0-9 무작위 값 구한 후, 4 이상이면 이동
- 경주를 완료한 후 우승자 판별

### 출력 처리

- 매 회차 마다 자동차 전진/이동 결과 출력
- 우승자 출력
    - 쉼표(`,`)로 구분

### 예외 처리

- 잘못된 값 입력 -> `IllegalArgumentException`
    - 자동차 이름이 공백
    - 시도 횟수가 0 이상 정수가 아님
