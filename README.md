# java-racingcar-precourse

1. 입력 요구사항 
    1. 자동차 이름 입력
       - (,) 로 구분해야함
       - 5자 이하
    2. 시도 횟수 입력
       - 양의 정수만 가능

2. 자동차
   1. 객체 생성
      - 이름
      - 이동거리
   2. 이동
      - 0, 9사이 무작위 함수 돌려서 4이상이면 이동거리 +1

3. 우승자 
   1. 자동차 이동거리 비교해서 최대값 가진 자동차 승자 배열에 추가
   2. 최대 값이 여러 명 일 수 있다

4. 출력 
   1. 우승자 객체 출력
      - 여러명이면 , 으로 구분
