# java-racingcar-precourse

# ⌨️ 입력
  - [ ] 자동차 이름들을 입력받는다
    - [ ] 6자 이상일 경우 예외 반환
  - [ ] 총 시도 횟수를 입력받는다
    - [ ] 정수값이 아닐 경우 예외를 반환한다

# 🖥️ 출력
- [ ] 각 단계별로 이동한 거리만큼 "-" 를 출력한다
- [ ] 최종 우승자를 출력한다

# ⚙️ 메인 로직
- [ ] 자동차게임 우승자를 뽑기 위한 필요한값을 입력한다
- [ ] 각 단계별로 자동차 게임을 진행한다
  - [ ] 각 자동차별로 랜덤한 값을 부여받는다
    - [ ] 값이 4 이상이면 한 칸 전진한다.
    - [ ] 값이 3 이하이면 현재 위치에 정지한다.
- [ ] 단계별로 진행한 자동차의 총 진행 거리를 각각 구한다
- [ ] 총 진행 거리를 비교하여 최종 우승자를 선정한다.
