# java-racingcar-precourse

## 구현할 기능 목록

- [x] 경주할 자동차 이름을 입력받는 기능

        - 콤마로 구분되어있나
        - 각각의 이름이 5자 이하인가
- [x] 시도할 횟수를 입력 받는 기능

        - 양의 정수인가
- [x] 전진 판단 기능

        - 0에서 9사이에서 무작위 값이 4 이상인 경우만 전진 
- [x] 실행 결과 출력 기능

        - 입력한 횟수만큼 랜덤 숫자뽑기 -> 전진판단 -> 결과출력 반복   
- [x] 최종우승자 판단 기능

사용자가 잘못된 값을 입력할 경우 ```IllegalArgumentException```을 발생시킨 후 애플리케이션은 종료되어야 한다.
