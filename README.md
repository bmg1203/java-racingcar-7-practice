# java-racingcar-precourse


## 기능구현
- [x] 자동차 이름 입력하기
- [x] 이동을 시도할 횟수 입력하기
- [x] 쉼표 구분자를 이용하여 입력한 자동차를 구별하고 배열에 넣기
- [x] 0~9 사이에 무작위 값 생성하기
- [ ] 무작위 값이 4 이상일 경우 전진하기
- [ ] 횟수만큼 시도할 때마다 전진한 횟수 출력하기
- [ ] 경주할 자동차 중 전진한 최대값 구하기
- [ ] 최대값을 가진 자동차를 최종 우승자로 선정하기
- [ ] 최종 우승자 출력하기
 

## 예외처리
- [ ] 자동차 이름은 6글자 이상이면 안됨
- [ ] 횟수는 1 이상이여야 함.
- [ ] 이외 예외처리
