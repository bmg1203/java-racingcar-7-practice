# java-racingcar-precourse


//구현 전
1. 자동차 이름 입력받기
2. 자동차 전진시키기(랜덤 활용)
3. 우승자 판별

//구현 중,후
1. 자동차 이름 입력받기

- 여러 대의 자동차 이름을 쉼표로 구분하여 입력받음
- 최대 5자로 제한

2.자동차 전진시키기

- 시도 횟수 입력받음, 그 수만큼 반복하기
- 각 자동차에 대해 0에서 9 사이의 무작위 값을 생성하고 그 값이 4 이상이면 한 칸 전진시킴

3. 우승자 판별

- 시도 횟수에 대한 전진이 끝나면 가장 멀리 전진한 자동차를 우승자로 판별
- 최대길이를 갖는 우승자들을 가져와 쉼표로 구분하여 출력
