# java-racingcar-precourse

RandomInput
- [ x ] 0 ~ 9 사이의 랜덤값을 구한다.
  - [ x ] 4 이상일 경우 전진한다.

InputView
- [ x ] 자동차 이름을 쉼표(,) 기준으로 구분한다.
- [ x ] 자동차 이름은 5자 이하로 한다.
- [ x ] 사용자는 이동할 횟수를 입력한다.

OutputView
- [ x ] 전진할 때 자동차 이름과 같이 출력한다.
- [ x ] 우승자를 출력한다.
    - [ x ] 우승자는 여려명 일 수 있다.
    - [ x ] 우승자는 쉼표(,)로 구분한다.

Car
- [ x ] 자동차는 전진 할 수 있다.
- [ x ] 자동차는 이름을 갖는다.

Exception
- [ x ] 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시킨 후 애플리케이션은 종료되어야 한다.
