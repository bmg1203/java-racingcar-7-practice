# 자동차 경주 🚗

## 구현할 기능 목록 ✅
- 경주할 자동차 입력받기
  - 이름은 쉼표 기준으로 구분
  - 이름은 5자 이하만 가능
- 시도할 횟수 입력받기
- 주어진 횟수 동안 자동차 전진, 멈춤 기능
  - 전진 조건 : 0에서 9 사이 무작위 값을 구하여 그 값이 4 이상일 경우
- 한 회차 수행 후 전진하는 자동차를 출력할 때 자동차 이름 함께 출력
  - 전진하는 자동차에 전진 횟수 만큼 " - " 출력
- 게임 완료 후 우승자 출력
  - 우승자 여러 명일 경우 쉼표(,)로 구분
- 사용자가 잘못된 값 입력시 `IllegalArgumentException` 발생 후 애플리케이션 종료
