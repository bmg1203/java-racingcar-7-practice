# java-racingcar-precourse

- 쉼표로 구분하여 자동차 이름 입력받기
	- 이름을 5자 이하로 제한
	- 잘못된 입력을 받으면 종료하기
- 자동차 이름을 쉼표를 기준으로 파싱
- 시도할 횟수 입력받기
	- 잘못된 입력을 받으면 종료하기(양의 정수로 제한)
	- 터미널에 출력되는 길이 제한하기
- 자동차 객체 생성
	- 이름저장
	- 이동을 판별할 수 있는 기능(숫자 생성 후 판별하기)
	- 실행 결과를 담을 String
- Application에 결과를 출력할 ViewPage 생성
	- 자동차 객체를 담을 list생성
	- 입력한 경기 종료 조건에 자동차 객체 중에 하나가 도달했는지 판정
	- 경기 종료 조건과 동일한 자동차 객체들을 승자로 결정
	- 쉼표로 구분해서 최종 우승자를 출력
