# java-racingcar-precourse

+ 자동차 이름 설정
    + 이름을 쉼표로 기준으로 구분
    + 이름은 5자 이하 !
        + 5자 초과하면 예외처리(huijun -> 예외처리)

+ 시도할 횟수
    + 사용자가 몇 번 이동을 할 것인지 입력
        + 0 을 입력하면 예외처리

+ 전진하는 조건
    + 무작위 값 구한 후 4 이상일 경우 전진
    + 4 미만이면 유지
    + "-" 으로 출력

+ 우승자 출력
    + 시도할 횟수가 끝난 후 누가 우승했는지 출력
    + 우승자는 한 명 이상일 수 있음
        + 쉼표(,)를 이용하여 구분

+ 테스트 케이스 작성
