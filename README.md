# 구현 기능 목록
- [x] 자동차 이름과 시도할 횟수를 입력받는다.
  - [x] 자동차 이름은 5자 이하만 가능하다.
  - [x] 시도할 횟수는 1회 이상이어야 한다.
  - [x] 시도할 횟수에는 숫자만 입력이 가능하다.
- [x] 입력받은 자동차를 생성한다.
  - [x] 자동차 개수가 2개 이상이어야 한다.
  - [x] 자동차 이름은 중복이 불가능하다.
- [x] 입력받은 횟수만큼 게임을 실행한다.
  - [x] 0~9 무작위 값 구한다.
  - [x] 4 이상일 경우 전진
- [x] 실행 결과를 출력한다.
- [x] 우승자를 구한다.
- [x] 우승자를 출력한다.
