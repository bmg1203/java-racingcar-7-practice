# java-racingcar-precourse

- [x] 자동차 이름을 입력 받는다.
    - 입력 올바르지 않으면 `IllegalArgumentException` 발생시킨 후 게임을 종료한다.
    - [ ] 쉼표기준으로 구분되어있지 않은 경우.
    - [x] 이름 5자 이하가 아닌 경우.
<br>

- [ ] 이동 횟수를 입력 받는다.
    - 입력 올바르지 않으면 `IllegalArgumentException` 발생시킨 후 게임을 종료한다.  
<br>

- [ ] 입력받은 횟수만큼 게임을 실행한다.
    - 자동차는 무작위 값에 따라 전진 또는 정지한다.
    - [ ] 0-9 사이 무작위 값을 생성한다.
    - [ ] 무작위값이 4 이상인지 확인한다.
    - [ ] 자동차가 전진한다.
<br>

- [ ] 게임 실행 결과를 출력한다.
    - 자동차의 위치는 문자 - 로 표시되어야 한다.
    - [ ] 각 자동차의 현재 위치를 구한다.
    - [ ] 현재위치를 문자 - 로 변환한다.  
<br>

- [ ] 우승자를 출력한다.
    - [ ] 우승자를 구한다.
    - [ ] 공동우승의 경우 쉼표로 구분한다.
