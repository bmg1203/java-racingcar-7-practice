# java-racingcar-precourse

## 구현 내용

1. 입력 
   - “경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)”를 출력
   - "시도할 회수는 몇회인가요?"를 출력
   - `camp.nextstep.edu.missionutils.Console.readLine()` 를 입력으로 설정
   - 입력한 내용에 대한 기본적인 예외 처리 적용 (빈 문자열, null, 숫자가 아닌 문자열 등)
   
2. 출력 
   - “실행 결과”를 출력
   - 각 자동차의 이름과 전진하는 거리를 시도 횟수만큼 출력
   - 최종 우승자를 출력
   
3. 참가 자동차 저장
   - ArrayList를 사용하여 참가할 자동차의 목록을 저장

4. 참가 자동차 문자열 처리
   - 입력한 문자열을 쉼표(,)를 기준으로 분리
   - 문자열의 길이가 5를 초과하거나 1 미만일 경우 `IllegalArgumentException` 발생
   - 문자열의 중복체크

5. 게임 실행
    - Random 값은 `camp.nextstep.edu.missionutils.Randoms`의 `pickNumberInRange()`를 사용

6. 테스트 케이스
    - 잘못된 입력에 대한 테스트 케이스 작성
      1. 빈 문자열
      2. 5자리 초과
      3. ,로 시작
      4. ,로 끝
    - 우승자에 대한 테스트 케이스 작성
      1. 우승자가 한 명일 경우
      2. 우승자가 두 명 이상일 경우
      3. 우승자가 없을 경우

7. 문서화
    - README.md 파일에 구현 내용을 정리
    - 각 메소드에 대한 설명 추가
    - 각 클래스에 대한 설명 추가
