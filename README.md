# java-racingcar-precourse

# 구현할 기능 목록

- ### Controller
- [ ] 경주할 자동차 이름 및 시도 횟수 입력 기능

- ### View
- [ ] 실행 결과 및 우승자 출력 기능

- ### Model
- [ ] 자동차 클래스 구현

  - ### Validator => Exception
    - [ ] 자동차 이름을 입력할 때 쉼표(,) 를 기준으로 나눴는지 확인하는 기능
    - [ ] 각 자동차의 이름 길이가 5자 이하인지 확인하는 기능
    - [ ] 시도 횟수에서 숫자 입력 외의 입력이 들어오진 않았는지 확인하는 기능
