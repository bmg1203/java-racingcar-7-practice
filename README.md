# java-racingcar-precourse

# 자동차 경주

## 기능소개

- 주어진 횟수안에서 확률에 따라 전진하는 자동차 경주 게임

## 패키지구조

- domain

- view
    - InputView
    - OutputView

## 기능목록

- [ ] 자동차 전진시키기
- [ ] 우승자 구하기

### 입력

- [ ] 쉼표로 구분된 경주할 자동차 이름 입력
    - [ ] 이름이 5자 이상일 경우 예외처리
    - [ ] 알파벳, 숫자 이외의 문자가 입력된 경우 예외처리
- [ ] 시도할 횟수 입력
    - [ ] 숫자를 입력하지 않은 경우 예외처리

### 출력

- [ ] 실행 결과 출력
- [ ] 우승자 출력
