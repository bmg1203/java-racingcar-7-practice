
### 기능 구현

### 1. 입력된 문자열에서 이름 추출 기능
  
 - [x] 쉼표를 기준으로 입력된 문자열에서 이름 추출
 - [x] 이름이 5자를 초과할 때 예외처리
 - [x] 입력에서 이동 횟수 추출
 - [x] 숫자 외 입력이 들어올 시 예외처리
   
### 2. 자동차 경주 기능

- [ ] 랜덤 값 생성기 구현
- [ ] 매 이동마다 각각의 자동차에 해당 값 부여
- [ ] 주어진 이동 횟수에 다다랐을 때 우승자 추출
   
### 3. 출력 기능
- [ ] 입력 안내 문구 출력
- [ ] 매 이동마다 각각의 자동차가 지나온 거리를 시각적으로 출력
- [ ] 최종 우승자 출력
