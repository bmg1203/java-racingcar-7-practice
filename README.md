---

## 🛠️ 구현할 기능 목록

### 사용자의 입력 받기
-  자동차 이름 문자열 받기
-  이동 횟수 받기
- camp.nextstep.edu.missionutils.Console의 readLine()을 활용

### 자동차 이름 파싱
- 쉼표 기준으로 파싱
- 이름이 5자 초과일 경우 에러 발생

### 경주 게임
- 전진 조건에 맞게 자동차 전진
- Map을 활용하여 자동차 이름에 맞는 전진횟수 추가
- 이동 횟수만큼 반복 후 우승자 추출
- camp.nextstep.edu.missionutils.Randoms의 pickNumberInRange()를 활용

### 출력
- 차수별 실행 결과 출력
- 우승자 출력
