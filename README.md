# java-racingcar-precourse
## 기능 목록
- 사용자 입력 받기
  - 경주할 자동차 이름 입력
  - 시도할 횟수 입력
- 유효성 검사
  - 유효하지 않은 경우 IllegalArgumentException 발생
- 자동차 생성 및 저장
  - Car 클래스
  - CarManager 클래스 
- 자동차 경주
  - RacingCarGame 클래스
- 진행 이력 출력하기
  - 자동차의 이름 출력
  - 이동 거리 출력
- 경주 결과 출력하기
  - 최종 우승자 출력
    - 여러 명일 경우 쉼표(,)로 구분
