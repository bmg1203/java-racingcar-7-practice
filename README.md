# 기능 목록

1. 사용자 입력 1 (자동차 이릅)
    - 자동차 이름을 입력 받는다.
    - 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하만 가능하다.
    - 선 출력 사항 : (경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분))
2. 사용자 입력 2 (시도 횟수)
    - 시도 횟수를 입력 받는다. (몇 번의 이동을 할 것인지)
    - 선 출력 사항 : (시도할 횟수는 몇 회인가요?)
3. 자동차 객체 생성
   - 이름과 전진 횟수를 가진 자동차 Class를 만든다.
   - 사용자 입력으로 받은 자동차 수만큼 자동차 객체를 생성한다.
