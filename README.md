# java-racingcar-precourse

## 기능 목록

### 입력

- [ ] 사용자 입력 받기
    - [x] n대의 자동차,자동차 이름으로 입력
    - [x] 쉼표를 통해 구분
        - [ ] 쉼표로 구분하기 전에 공백을 제거한다
    - [ ] 시도 횟수(라운드 수) 입력

### 도메인

- [ ] 자동차
    - [ ] 전진
    - [ ] 전진 조건 판단(랜덤 값 4 이상일 때 전진)
    - [ ] 위치 계산
- [ ] 값 생성
    - [ ] 랜덤 값 생성
- [ ] 참가자들
    - [ ] 우승자 판단

### 컨트롤러

- [ ] 라운드 진행

### 출력

- [ ] 자동차 이름
- [ ] 라운드 진행 후 위치 상태
- [ ] 우승자 출력
    - [ ] 2명이상일 시 ,를 기준으로 구분

### 예외 처리

- [ ] 자동차 이름이 5자를 초과
