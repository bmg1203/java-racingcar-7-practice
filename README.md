# java-racingcar-precourse

## 기능 목록

### 1. 자동차 경주 게임 시작

- 사용자가 경주할 자동차의 이름을 입력받는 기능
- 게임 라운드 수를 입력받는 기능
- 잘못된 값을 입력한 경우 에러를 발생시키고 애플리케이션을 종료하는 기능

### 2. 자동차 경주 게임 진행

- 0~9 사이의 무작위 값을 생성하여 자동차의 전진 여부를 결정하는 기능
  - 4 이하이면 자동차를 멈춘다.
  - 4 이상이면 자동차를 전진시킨다.
- 각 라운드 별로 게임 진행 결과를 출력하는 기능

### 3. 자동차 경주 게임 종료

- 게임이 완료된 후 우승자를 출력하는 기능
  - 우승자가 여러 명일 경우를 처리한다.
  - 우승자가 한 명일 경우를 처리한다.
