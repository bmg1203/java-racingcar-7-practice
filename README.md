## 기능목록 (구현항목, 예외처리)

### 구현항목
1. 사용자 입력하라고 알려주기
2. 사용자 입력 받기(자동차이름, 시도할 횟수)
3. 자동차 이름 "," 기준으로 구분하기
4. 자동차 이름 5글자 이하인지 확인하기
5. 각 사람 차례마다 Randoms API 사용하여 결과값이 4이상인지 확인하기
6. 4이상일때 전진시키기
7. 차수별 실행 결과 출력하기(이때 이름과 전진횟수 출력)
8. 우승자 구하기
9. 우승자 여러명일 경우 "," 이용하여 구분하기
10. 우승자 출력하기

### 예외처리
* 사용자가 요청에 맞게 입력했는지 확인
