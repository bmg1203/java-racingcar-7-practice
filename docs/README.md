# 미션 - 자동차 경주 미션

## 📋 구현 기능 목록

### 자동차 입력단계

- [ ] 자동차 입력을 위한 문구출력
- [ ] 자동차 이름 입력을 위한 문자열 입력
  - [ ] ","로 구분
  - [ ] 이름이 공백일경우 예외처리
  - [ ] 자동차의 이름의 길이는 1 ~ 5가 아닐경우 예외처리
  - [ ] 자동차의 이름은 중복일경우 예외처리

### 시도할 횟수(라운드) 입력단계

- [ ] 시도할 횟수 입력을 위한 문구출력
- [ ] 시도할 횟수 입력기능
  - [ ] 숫자값 아닐경우 예외처리
  - [ ] 최소 값 1이 아닐경우 예외처리

### 경주 진행단계

- [ ] 라운드의 횟수 만큼 전진 또는 멈춤
- [ ] Randoms.pickNumberInRange(0, 9)의 값이 4 이상이면 전진 미만일경우 멈춤
- [ ] 전진 횟수만큼 '-' 로 출력

### 경주 결과단계

- [ ] 가장 많은 전진횟수를 가진 자동차가 우승자 
- [ ] 우승자가 여러 명일 경우 쉼표(,)를 이용하여 구분
